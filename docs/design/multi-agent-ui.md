## Event Bus Events

New events for multi-agent UI updates:

| Event | Payload | Description |
|-------|---------|-------------|
| `agent.switched` | `{ agentId, identity }` | Active agent changed |
| `agents.list` | `[{ id, identity, sessionCount }]` | Agent list updated |
| `node.connected` | `{ nodeId, name, url }` | Remote node connected |
| `node.disconnected` | `{ nodeId, reason }` | Remote node disconnected |
| `swarm.progress` | `{ swarmId, completed, total }` | Swarm task progress |

## Accessibility Considerations

- Agent badges include `aria-label` with full agent name
- Sub-agent cards are collapsible with `aria-expanded`
- Node status uses both color AND icon for colorblind users
- Split view panes are keyboard-navigable (Tab to switch)
- Swarm progress includes screen-reader announcements
